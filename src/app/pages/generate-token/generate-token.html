Как сгенерировать токен для входа?

  1. Перейдите на профиль вашего аккаунта GitLab
  2. В правом верхнем углу кликните на иконку профиля
  3. Кликните на Preferences
  4. В левой колонке кликните на Access
  5. Выберите Personal Access Token
  6. Generate Token
  7. Legace Token
  8. Введите имя токена
  9. Выберите read_user и read_api
  10. Generate token
  11. Copy token
  12. Done!
